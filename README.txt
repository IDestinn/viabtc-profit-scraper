ViaBTC PPS Profit Parser
========================

Назначение
-----------
Программа собирает данные PPS Profit из ViaBTC API
для монет:
- BTC
- LTC
- KAS

и сохраняет результат в Excel-файл output.xlsx.


Как использовать
-----------------

1. Поместите файл:
   input.csv

   рядом с программой:
   viabtc-profit-scraper.exe


2. Формат input.csv:

   name;link
   Ваня;https://viabtc.com/....

   Пример:

   name;link
   Client 1;https://www.viabtc.com/observer/dashboard?access_key=XXXXX
   Client 2;https://www.viabtc.com/observer/dashboard?access_key=YYYYY


3. Запустите:
   viabtc-profit-scraper.exe


4. Программа спросит диапазон дат.

   Если нужен весь период:
   просто нажмите ENTER два раза.

   Пример:

   Дата начала диапазона (YYYY-MM-DD):
   Дата окончания диапазона (YYYY-MM-DD):


   Пример с диапазоном:

   Дата начала диапазона (YYYY-MM-DD): 2026-05-01
   Дата окончания диапазона (YYYY-MM-DD): 2026-05-31


5. После завершения появится файл:
   output.xlsx


Что содержит output.xlsx
------------------------

Для каждого клиента выводится:

- Имя клиента
- Ссылка
- Монета
- PPS Profit
- Сумма по всем монетам клиента

Также внизу файла:
- Общая сумма (общая сумма всех клиентов и всех монет)


Поддерживаемые монеты
---------------------
- BTC
- LTC
- KAS


Важно
------
- Для работы нужен интернет.
- Не изменяйте access_key в ссылках.
- Если ViaBTC API временно недоступен,
  программа покажет ошибку в консоли.


Файлы
------
input.csv   - входные данные
output.xlsx - результат работы
